1. CLI seeders
   - Khởi chạy.
      - npx sequelize-cli db:seed:all
   - Thu hồi trước đó.
      - npx sequelize-cli db:seed:undo:all

2. CLI migrations
   - Câu lệnh chạy lại migrations
      - npx sequelize-cli db:migrate
      - npx sequelize-cli db:migrate:undo:all
   - Câu lệnh xóa và tạo lại db
      - npx sequelize-cli db:drop
      - npx sequelize-cli db:create

5. Lệnh tạo liên kết
      hasMany(1-n), belongsTo(n-1), belongsToMany(n-n), hasOne(1-1) 
      Employee.belongsToMany(models.Trip, {
            through: models.TripEmployee,
            foreignKey: "employee_id",
            otherKey: "trip_id",
            as: "employee_to_trip",
         });
      Trip.belongsToMany(models.Employee, {
            through: models.TripEmployee,
            foreignKey: "trip_id",
            otherKey: "employee_id",
            as: "trip_to_employee",
         });
